HTML

HTML은 Hyper-Text Mark-up Language의 준말로 하이퍼텍스트 형식의 웹문서를 만들 수 있는 웹 언어이다. 하이퍼 링크를 통해서 서로 다른 문서로 이동할 수 있다. 국제적 표준으로 자리잡았고 현재는 HTML5까지 개발되었다.

CSS

CSS는 Cascade Style Sheet의 준말로 웹문서에 스타일을 적용할 수 있는 시트이다. 부모 요소의 스타일을 자식요소에 상속하는 특징이 있으며 선택자를 통해서 다양한 방법으로 스타일을 적용할 대상들을 자동 지정할 수 있다.

Javescript

자바스크립트는 html과 css로 만들어진 클라이언트 웹문서를 콘트롤할 수 있는 인터프리터 스크립트 언어이다. 별도의 컴파일을 필요로 하지 않으며 현재의 대부분의 브라우저에서 정상적으로 작동한다.
